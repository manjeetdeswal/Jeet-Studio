Android Essential Plugin Documentation

This guide provides a reference for all Blueprint nodes across the plugin suite.

PAK File Utilities

Mount Pak File (Async)

Mounts a .pak file asynchronously without freezing the game. This is useful for loading DLC, mods, or extra content at runtime. The node will execute one of its output pins when the operation is complete.

Pak File Path (In): String - The absolute path to the .pak file on the device.
On Success (Out): Exec - Fires when the PAK is successfully mounted. Provides a success message.
On Failure (Out): Exec - Fires if the mounting fails. Provides an error message.

Unmount Pak File

Unmounts a previously mounted .pak file, making its assets unavailable and freeing up resources.

Pak File Path (In): String - The absolute path to the .pak file to unmount.
Return Value (Out): Boolean - Returns true if the unmount was successful or if the file wasn't mounted.
Out Message (Out): String - Provides information about the result of the operation.

Android App Management

Check For Update

Initiates the Google Play In-App Update flow. If an update for your app is available on the Play Store, Google Play will display a dialog to the user, allowing them to update the app immediately or flexibly in the background.

Testing Requirement: This will only work with a build that has been uploaded to the Google Play Console (e.g., Internal Testing track). You must install this build, and then upload a newer version to the same track to trigger the update prompt.

Request Review

Initiates the Google Play In-App Review flow. This will show a review dialog to the user inside the app. The dialog's appearance is controlled by a quota set by Google to prevent spam.

Usage Guidelines: Do not call this from a button (e.g., "Rate App"). Instead, trigger it after a positive user experience, such as completing a level or achieving a high score. The dialog may not appear every time you call the function. Testing requires a build from the Google Play Console.

Media & Sharing

Share Game Screenshot

Captures the current screen and opens the native Android share dialog.

Chooser Title (In): String - The title of the share dialog (e.g., "Share Screenshot").
Note: This requires a rendered world. It will produce a black image if called from a context without a 3D viewport (like a UI-only widget).

Save Game Screenshot to Gallery

Captures the current screen and saves it directly to the device's public "Pictures" gallery.

File Name (In): String - The desired name of the file, without the extension.

Share Image

Shares a `Texture2D` asset from your project via the native Android share dialog.

Image (In): Texture 2D - The texture asset to share.
Chooser Title (In): String - The title of the share dialog.

Share Image From Path

Shares a local image file from a given path.

Image Path (In): String - The absolute path to the image file on the device.
Chooser Title (In): String - The title of the share dialog.

Share Text

Shares a simple text string via the native Android share dialog.

Text (In): String - The text content to share.
Subject (In): String - The subject line, used by apps like email clients.

Device & Hardware

Is Internet Connected

Returns true if the device has an active network connection with internet access.

Return Value: Boolean - True if connected, false otherwise.

Get Battery Percentage

Returns the device's current battery level as a percentage.

Return Value: Integer - A value from 0 to 100.

Get SDK Version

Returns the Android API level of the device (e.g., 33 for Android 13).

Return Value: Integer - The Android API level.

User Interface

Show Toast

Displays a small, temporary pop-up message (a "toast") at the bottom of the screen.

Message (In): String - The text to display.
Long Duration (In): Boolean - If true, the toast lasts longer.

Show Time Picker

Opens the native Android time selection dialog. Triggers the `On Result` event when the user picks a time or cancels.

On Result (In): Event - An event that fires with the result. You must create a custom event and connect it to this pin.
Usage: Right-click the `On Result` pin and select "Create Custom Event". This will create a new event node with `Hour`, `Minute`, and `bSuccess` output pins.

Show Date Picker

Opens the native Android date selection dialog. Triggers the `On Result` event when the user picks a date or cancels.

On Result (In): Event - An event that fires with the result.
Usage: Right-click the `On Result` pin and select "Create Custom Event". This creates a new event node with `Year`, `Month`, `Day`, and `bSuccess` outputs.

Pick From Gallery

Opens the native Android gallery to let the user select an image, video, or audio file.

On Result (In): Event - An event that fires with the result.
Usage: Create a custom event from the `On Result` pin. It will provide the `File Path`, `File Name`, `bSuccess`, and a loaded `Texture 2D` (if the selected file was an image).

Pick Any File

Opens the native Android file browser for selecting any type of file.

On Result (In): Event - An event that fires with the result.
Allow Multiple (In): Boolean - If true, the user can select multiple files.
Usage: Create a custom event from the `On Result` pin. It will provide arrays of file information.

Launch URL Fixed

Opens the specified URL in the device's default web browser.

URL (In): String - The web address to open.

Permissions

Check And Request Android Storage Permissions

Requests `READ_EXTERNAL_STORAGE` and `WRITE_EXTERNAL_STORAGE` permissions. This is for older Android versions (below API 30).

On Permission Granted (Out): Exec - Fires if all permissions are granted.
On Permission Denied (Out): Exec - Fires if any permission is denied.
Permissions (Out): Array of String - The list of granted permissions.

Request Notification Permission

On Android 13 (API 33) and newer, you must request permission to post notifications.

On Permission Granted (Out): Exec - Fires if permission is granted.
On Permission Denied (Out): Exec - Fires if permission is denied.

Is Notification Permission Granted

Returns true if the permission to post notifications has been granted.

Return Value: Boolean - True if permission is granted.

Notifications

Post Notification

Displays a standard Android notification in the status bar.

Title (In): String - The title of the notification.
Content (In): String - The main text of the notification.
Notification Id (In): Integer - A unique ID for this notification.

Post Notification with Texture Icon

Displays a notification with a large icon provided by a `Texture2D` asset.

Title (In): String
Content (In): String
Icon Texture (In): Texture 2D - The texture to use as the large icon.
Notification Id (In): Integer
Important: For this to work, the texture asset must have its Compression Settings set to UserInterface2D (RGBA).

Cancel Notification

Removes a specific notification from the status bar.

Notification Id (In): Integer - The ID of the notification to cancel.

Cancel All Notifications

Removes all notifications posted by your app.
